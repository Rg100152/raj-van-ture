🌿 Raj Van Ture — Wildlife Portal

Raj Van Ture is a modern, dark-themed wildlife exploration and ticket booking web application.
It provides an immersive experience for users to explore wildlife, book safari tickets, and generate digital receipts.

---

🚀 Features

- 🌌 Dark-themed cinematic UI
- 🐅 Wildlife gallery (20+ animals)
- 🎟️ Ticket booking system
- 💰 Automatic price calculation
- 💳 Simulated payment gateway (UPI/Card)
- 🧾 Digital receipt generation + print option
- 📱 Fully responsive design

---

🛠️ Tech Stack

- HTML5
- CSS3 (Advanced styling + animations)
- JavaScript (DOM + logic handling)
- Font Awesome (icons)
- Google Fonts
